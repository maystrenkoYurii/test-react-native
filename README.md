# TestReactNative App

## RUN DEVELOPMENT:
### 1. Install dependencies:
```
npm install
```
### 2. Run packager:
```
npm run start
```
### 3. Run application:
```
npm run android || npm run ios
```

## RUN SCRIPTS:
### 1. Unit test:
```
npm run test
```
### 2. ESLint test:
```
npm run lint
```
### 3. Full test:
```
npm run test:full
```
### 4. Format code:
```
npm run format
```
